Additional Info

Field Value
Description

Point file of all ball diamonds located in City of Ottawa parkland. File includes baseball, softball and t-ball diamonds.
Date First Published
Update Frequency
As required.
Accuracy

Points were collected in the field using GPS software on handheld computers. Points were then verified / adjusted using aerial photography.
